# Food Tracker

Веб-проект для отслеживания приёмов пищи и подсчёта процента дневной нормы калорий.

Онлайн-версия приложения:
https://food-tracker-nutrihack.vercel.app/

## Технологии

React (хуки: useState, useEffect)

Vite — сборка проекта

CSS Modules — модульные стили

JSON Server — mock backend (REST API)

Fetch API — работа с сервером

Vercel — деплой приложения

## Функциональность

- Отображение списка приёмов пищи
- Добавление нового приёма пищи через модальное окно
- Получение списка приёмов пищи с сервера (GET)
- Отправка нового приёма пищи на сервер (POST)
- Подсчёт процента дневной нормы калорий (из расчёта 2000 ккал)

Для хранения данных используется JSON Server.

Запуск backend:

```bash
npx json-server --watch db.json --port 3001

Запуск проекта:

npm install
npm run dev
