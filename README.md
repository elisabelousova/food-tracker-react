**Трекер питания (React)**

Учебный веб-проект для отслеживания приёмов пищи и потреблённых калорий.
Проект реализован на React с использованием современного сборщика Vite.

**Используемые технологии**

•	React

•	Vite

•	JavaScript 

•	CSS Modules

•	React Hooks (useState)

**Как запустить проект**

npm install

npm run dev

**Примечание**

Модальное окно добавления приёма пищи реализовано как заглушка.
Основной фокус задания — архитектура React-приложения.
